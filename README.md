## Create a repo

Create a local folder
```bash
git init LOCAL_FOLDER
```

Clone the repo from github to local folger
```bash
git clone GITHUB_REPO LOCAL_FOLDER
```

## Update a repo

Add to the changes
```bash
git add --all
```

Create a commit with name
```bash
git commit -m "NAME OF COMMIT"
```

Push the commit to github
```bash
git push
```
